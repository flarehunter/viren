Past, present & future

Knowledge (& especially legislation) is constantly in motion. Viren allows you to not only manage past & present knowledge but also future or expected knowledge. Every change in your knowledge model is captured & we never remove past versions.

Time to market

Viren allows you to reduce project lead time by working in parallel. Your domain expert can start modelling while your developer starts implementing the application. The knowledge model also combines analysis & development time because it is executable.

Simulate & optimize

We want to assist in decision making. This requires evaluation of a knowledge model from different angles by simulating. Maybe you want to evaluate different models side by side? No problem. Or even better: Let us calculate the optimal inputs towards a certain result.

Performance

The Viren engine is optimized for speed. Speed is a prerequisite for fast simulation & optimization. A single instance of the execution engine can handle more than 10k calculations per second. And we can scale using Azure cloud resources when you momentarily need more throughput.

Transparancy

We save every calculation that has ever been executed. And we support troubleshooting by combining the knowledge model with the actual calculation data. This provides 100% transparency on every expression & every parameter in the calculation.
